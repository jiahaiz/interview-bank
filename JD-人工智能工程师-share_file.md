## 岗位职责

1. **AI Agent 系统**：负责公司 AI Agent 智能体系统的架构设计、开发与工程化落地，构建具备自主规划、工具调用、记忆管理和多步推理能力的智能体应用。
2. **多智能体协同**：设计和实现多智能体协同系统，包括 Agent 间的任务分配、通信机制、工作流编排，支撑复杂业务场景的自动化决策与执行。
3. **LLM 与能力建设**：基于大语言模型进行 Prompt 工程、Function Calling、RAG 检索增强生成、Fine-tuning 等方向的研发，持续优化 Agent 的推理质量和任务完成率。
4. **云与 AI 服务**：基于阿里云（通义系列、PAI、百炼、OSS、ECS、函数计算 FC、向量检索等）构建和优化 AI 服务架构，保障高可用和高性能。
5. **IaC / CI/CD**：使用 CDK（Cloud Development Kit）及 IaC 工具，完成云上资源的自动化定义、部署与管理，搭建可复现的 CI/CD 流水线。
6. **Agent 工具链**：开发和维护 Agent 所需的工具链（Tool/Plugin），包括 API 集成、数据库查询、文件处理、外部系统对接等。
7. **数据与模型 Pipeline**：设计并实现数据采集、清洗、特征工程及模型训练的完整 Pipeline，持续迭代优化模型效果。
8. **文档与规范**：编写高质量的技术文档与接口文档，参与技术方案评审与 Code Review，推动团队工程规范建设。
9. **技术预研**：关注 AI Agent 前沿技术动态，进行技术预研和可行性验证，输出技术选型建议。

## 任职要求

### 基本条件

- 本科学历 + **3 年及以上**相关工作经验，或**硕士学历 + 1 年及以上**相关工作经验（AI、计算机、数学、统计等相关专业优先）。
- 有扎实的计算机基础，熟悉数据结构、算法和常用设计模式。

### AI Agent 开发能力

- 熟练掌握至少一种主流 Agent 开发框架：**LangChain、LangGraph、CrewAI、AutoGen、MetaGPT、Dify、Coze** 等，有完整的 Agent 项目落地经验。
- 深入理解 Agent 核心机制：**ReAct** 推理模式、**Planning** 规划、**Tool Use** 工具调用、**Memory** 记忆管理、**Reflection** 反思与自纠错等。
- 具备**多智能体（Multi-Agent）**系统的设计与开发经验，能根据业务需求设计合理的 Agent 角色分工与协作流程。
- 熟悉 **Function Calling** 和 **Tool Integration** 的实现方式，能高效开发自定义工具并集成到 Agent 工作流中。
- 了解 Agent 评估与可观测性方案，能对 Agent 行为进行追踪、调试和效果评测。

### 技术能力

- **精通 Python**，熟练使用至少一种深度学习框架（**PyTorch、TensorFlow、PaddlePaddle**）。
- 熟悉**阿里云生态**：ECS、OSS、NAS、RDS、SLS、函数计算 FC、PAI（DSW 和 EAS）、DashScope、百炼平台、向量检索 **Lindorm** 和 **Elasticsearch** 等，有实际项目使用经验。
- 熟悉 **CDK、Terraform、Pulumi** 等 IaC 工具，具备云基础设施自动化部署和管理经验，了解 **DevOps** 和 **MLOps** 最佳实践。
- 熟悉 **RAG** 技术栈：文档解析、分块策略、Embedding 模型选型、向量检索优化、重排序等。
- 熟悉 **Docker、Kubernetes** 等容器化技术，具备模型服务化部署经验（如 **Triton、vLLM、TGI**）。
- 熟悉常用机器学习和深度学习算法（**NLP、CV、推荐系统**至少一个方向有深入实践）。

### 加分项（满足任一即可）

- 有大模型微调（**LoRA、QLoRA、SFT、RLHF、DPO**）实战经验。
- 有将 Agent 系统应用于**实际生产环境**的经验（如智能客服、自动化运维、数据分析助手、代码生成助手等）。
- 熟悉向量数据库（**Milvus、Qdrant、Chroma、Elasticsearch kNN**）及嵌入模型调优。
- 有多模态模型（图文理解、语音识别等）项目经验。
- 了解模型量化、推理加速、边缘部署等模型优化技术。
- 有开源项目贡献经验或技术博客输出，**GitHub 活跃者优先**。
- 具备良好的**英文技术文献阅读能力**。

### 软素质

- 较强的自驱力与学习能力，对 AI 新技术保持热情和敏锐度。
- 良好的沟通协作能力，能与产品、业务团队高效对接需求。
- 具备独立分析问题和解决复杂技术难题的能力。
- 有良好的工程意识，注重代码质量与系统可维护性。
